About RunRelease

RunRelease is an innovative iOS app that helps users build healthier phone habits by unlocking selected apps only after completing GPS-tracked runs. The app uses Apple's Family Controls API to provide a seamless, system-level app management experience.

Technical Implementation

Family Controls API: The app leverages Apple's Family Controls framework to provide system-level app management. This allows RunRelease to lock and unlock apps based on user activity, providing a seamless experience that integrates with iOS's built-in Screen Time features.

GPS Tracking: Uses Core Location for accurate run tracking, ensuring users complete genuine physical activity before unlocking their selected apps.

Privacy & Security: All user data is stored locally on the device. The app requires Family Controls entitlement approval from Apple to ensure proper system integration and user safety.
